Realtime Angry Admin

Realtime Framework multi-player mobile game example

Every developer knows that System Administrators can easily lose their temper and be a pain in the neck. Can't live with them but seriously can't live without them either. This is our homage to them and in particular to our own SysAdmins team leader at Realtime.

In this game you can unleash your Angry Admin and throw him to make damage to your opponent friends. Players lose when the rock falls over their cannons.

Every player has a game Id that identifies their game (4 letters). Give yours to your friends so they will be able to play with you. You may also invite your friends by inserting their game Ids.

This game acts as an example to show how you can use the Realtime® Framework to build a real-time multi-player mobile game. In this game we use both of the Realtime® Framework products: STORAGE (Realtime Cloud Storage for data persistence, like the scores) and MESSAGING (Realtime Cloud Messaging for the game real-time syncing between players). We also make use of Cocos2D (cocos2d-iphone is a framework for building 2D games) and Sprite Builder

For more details check the complete source code at GitHub and the development guide tutorial blog post.


May the best Angry Admin win.
